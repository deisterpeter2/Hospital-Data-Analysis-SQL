# hospital-data-analysis-sql
This project demonstrates SQL skills through the analysis of hospital patient and operational data. The project includes database creation, data loading, data cleaning, data transformation, and analytical queries designed to generate business insights.

Objectives
-- Create and populate a relational database
-- Clean and validate imported data
-- Analyze hospital data using SQL
-- Identify trends and key performance metrics
-- Demonstrate common SQL techniques used in data analyst roles

Skills Demonstrated
-- SQL Database Design
-- Data Cleaning and Validation
-- Data Transformation
-- Data Analysis
-- Business Reporting

SQL Concepts Used
-- INNER JOIN 
-- Subqueries
-- Common Table Expressions (CTEs)
-- Aggregate Functions (SUM, AVG, COUNT)
-- Window Functions
-- Ranking Functions
-- CASE Statements
-- Date Functions
-- Data Type Conversion

Example Business Questions Answered
-- Which departments treated the highest number of patients?
-- What was the average patient length of stay by department?
-- Which physicians handled the highest patient volume?
-- What were the monthly admission trends?
-- Which departments generated the highest revenue?

Files included:
-- Hospital_Create_Table.sql (Queries used to create tables)
-- Hospital_Data_Cleaning.sql (Queries used to clean data)
-- Hospital_Data_Analysis.sql (Queries used to analyze data)
-- Hospital_Insert_PatientVisits_Data.sql, Hospital_Insert_Patient_Doctors_Data.sql, Hospital_Insert_Dept_Treatment_Diagnosis_PayMethod_Data.sql (Queries used to load data)
